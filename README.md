## About Project
### 开发环境
- JDK(>=1.8)
- Maven(>=2.x)
- IDEA(best)

### 技术说明
1. 开发框架: Spring boot -> Spring cloud
2. 缓存方案: Redis为主(分布式缓存及会话共享)
3. 权限框架: Spring security
4. 程序监控: Metrics、Hystrix(服务熔断、降级等)
5. 数据源: HikariCP
6. 在线文档: Swagger
7. 消息总线: Kafka
7. ...
