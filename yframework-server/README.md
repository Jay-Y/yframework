## About Project
### 开发环境
- jdk(>=1.8)
- maven(>=2.x)
- IDEA(best)

### 技术说明
1. 架构思想: DDD(领域驱动设计)
2. 开发框架: Spring boot -> Spring cloud
3. 缓存方案: redis为主(分布式缓存及会话共享)
4. 权限框架: Spring security
5. 程序监控: Metrics、Hystrix(服务熔断、降级等)
6. 数据源: HikariCP
6. 在线文档: Swagger
7. ...
